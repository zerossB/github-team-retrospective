üìù How to Apply a Theme

To use one of these themes, edit the :root CSS section in templates/report_template.html:

:root { /* Replace with your chosen theme colors */ --color-primary: #5b6efc; --color-primary-dark: #4557e0; --color-primary-light: #8b9aff; /* Other colors remain the same */ --color-secondary: #ff6b6b; --color-accent: #ffd93d; /* ... etc ... */ }

For example, to use the Green theme, replace the primary colors with:

:root { --color-primary: #51cf66; --color-primary-dark: #2f9e44; --color-primary-light: #8ce99a; /* ... rest of the colors ... */ }

After making the change, regenerate your reports to see the new colors!
